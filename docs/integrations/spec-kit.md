# MOD-W with GitHub Spec Kit

> See the full integration guide: [`articles/modw-with-spec-kit.md`](../../articles/modw-with-spec-kit.md)

GitHub Spec Kit is an open-source SDD toolkit for capturing constitutions, specs, plans, and tasks in your repo. MOD-W layers on top — defining who approves each phase, introducing intentional role and model contrast, and enforcing cross-review before any phase is accepted.

See [`README.md`](../../README.md) for the Spec Kit section, and [`articles/`](../../articles/) for the detailed playbook.
