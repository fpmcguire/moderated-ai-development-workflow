# MOD-W with Kiro

> See the full integration guide: [`articles/modw-with-kiro.md`](../../articles/modw-with-kiro.md)

Kiro provides a spec-driven IDE runtime. MOD-W layers governance on top — Moderator authority, role separation, and cross-agent review discipline. Kiro's Requirements / Design / Tasks phases map directly to MOD-W's `PRODUCT.md`, `ARCHITECTURE.md`, and `ROADMAP.md` + `STEP-XX.md`.

See [`README.md`](../../README.md) for the Kiro section, and [`articles/`](../../articles/) for the detailed playbook.
